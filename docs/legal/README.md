# Legal documentation

This folder contains the version-controlled source legal documentation pack for Nineteen Point Two Limited and WAIA, the Workplace AI Academy.

The public legal pages on the Nineteen Point Two website are the customer-facing versions of this material.

The DOCX file is retained as a source copy for version history, review and future legal or customer due diligence work.

When public legal pages are updated, changes should be checked against the current source pack in this folder.

Do not add secrets, credentials, environment values or confidential customer data to this folder.
